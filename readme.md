## ✦ Stack technique

| Couche | Technologie |
|--------|-------------|
| Structure | HTML5 sémantique |
| Style | CSS3 — variables, grid, animations |
| Interactions | Vanilla JavaScript |
| 3D / Canvas | Web Canvas API |
| Fonts | Syne + DM Mono (Google Fonts) |
| Hébergement | GitHub Pages |

Aucune dépendance externe. Aucun framework. Aucun bundler.

---

## ✦ Fonctionnalités

- **Canvas animé** — cubes 3D isométriques en fond qui tombent en continu
- **Curseur custom** — point bleu + anneau avec effet de lag
- **Cube 3D hero** — tourne lentement, s'explose en cartes au scroll
- **Carrousel auto** — défilement automatique avec pause au hover + image reveal
- **Globe rotatif** — apparaît au scroll, carte de navigation au hover
- **Système d'avis** — 3 statiques + formulaire avec persistance storage
- **Scroll reveal** — animations d'apparition au défilement
- **Footer impactant** — nom en grand en ghost text
